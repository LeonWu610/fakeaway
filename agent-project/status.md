## Current state
- “超市便利”和“蔬菜水果”现在是独立筛选键，不再依赖商家全文是否包含分类名称；前者只命中 `market`，后者只命中 `fruits`。
- 首页快捷分类与筛选弹层共用同一套明确品类规则；“全部零售”仍可统一查看超市、水果和健康用品。
- 新增全局购物车上下文，使用 `fakeaway.carts` 在 localStorage 按店铺保存商品；离开商家页、切换底栏页面或刷新后仍可回看。
- 底部导航新增“购物车”入口与总数量角标；商家页顶栏也可直接进入全局购物车。
- `/cart` 按店铺展示已选商品，支持增减数量、清空单店、清空全部、回店继续选和单店结算；不同店铺分别保留、分别结算。
- 店内原有购物袋面板继续使用同一份全局状态；空购物袋图标会打开全局购物车，不再出现无反馈点击。
- 确认下单后只清除对应店铺购物车，不影响其他店铺已经挑好的商品。
- 确认订单进入等待、等待进入送达、送达离开都使用完成态替换导航，不再把已经结束的步骤留在正常返回链路中。
- 从送达页“回店里再看看”进入店铺后，店铺左上角返回会直接回首页，不再返回送达页。
- 全屏送达庆祝使用按订单的 sessionStorage 标记；同一订单重新挂载或刷新时只显示普通送达内容，不重复播放“到了”动画。
- “收藏这份记忆”增加局部弹跳、爱心/星光散开和短提示；取消收藏不播放庆祝，再次收藏可重新触发。
- 上一轮已验收的下单、任务完成、送达动画，以及筛选遮挡、无响应按钮、等待计时和文案修复均需保留。
